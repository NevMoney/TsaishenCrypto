Why is decentralization important?

With a centralized network it is easy to shut it down. If a government where you live does not like the project or the server provider goes down, everything disappears. Using publicly available network that is decentralized, even if one server goes down or one government shuts down the project, it still exists on thousands of different computers, thus making it impossible to remove.

Why is blockchain important?

Blockchain uses cryptography to encrypt the data and thusby secure it. All of the information (data) is distilled into a cryptographic hash, which is a function that takes all data and turns it into a fixed-size alphanumeric string. Whenever the data changes, the hash string changes, so you know if someone has altered any information. When a new block is created it uses the hash from the previous block and appends new information. With this process, it is impossible to change any information without everyone knowing that things have changed. This makes blockchain the absolute safest database on the planet.
